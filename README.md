### Controls

Up/Down/Left/Right arrow keys = LEM movement <br>
Shift/Space = Down/Up LEM movement <br>
Q/E = Rotate left/right LEM movement <br>

1 = LZ1 cam <br>
2 = LZ2 cam <br>
3 = LZ3 cam <br>
4 = Lander onboard cam <br>
5 = Top-down Lander cam <br>
6 = Below Lander cam <br>

` = Toggle tracking cam <br>
R = EasyCam wide view <br>
T = EasyCam wide view, lander position <br>


U = Restart on landing/death <br>
Z = Toggle Altitude sensor <br>
A = Toggle Fuel/Altitude HUD <br>
